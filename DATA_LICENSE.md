# Data License

This repository includes a small local classroom-demo subset of the Twin-2K-500
dataset under:

- `data/twin_2k_500_local/full_persona.parquet`
- `data/twin_2k_500_local/wave_split.parquet`
- `tmp/zh_translation_cache.json`

The Twin-2K-500 dataset is licensed under Creative Commons Attribution 4.0
International (CC BY 4.0).

Dataset source:
https://huggingface.co/datasets/LLM-Digital-Twin/Twin-2K-500

Documentation:
https://digital-twin-simulation-version2.readthedocs.io/

Paper:
https://arxiv.org/abs/2505.17479

The translation cache is a derived convenience artifact for the classroom demo
and should be treated under the same attribution requirements as the dataset
content it was generated from.

When reusing the bundled data or derived translations, provide attribution to
the Twin-2K-500 dataset authors and preserve the CC BY 4.0 notice.
